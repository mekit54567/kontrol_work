# Практические занятия — Информационная безопасность и защита информации

Репозиторий содержит учебные отчёты, скрипты и примерные результаты для выполнения практических заданий по курсу _«Информационная безопасность и защита информации»_.
Материалы подготовлены в учебных целях. Все действия воспроизводимы локально на ваших виртуальных машинах (Metasploitable, Ubuntu, Kali). 

**Структура:**
- `REPORT.md` — общий отчёт по всем заданиям.
- `actions/` — подробные поэтапные отчёты (Nmap, SSH brute-force, HTTP Basic).
- `scripts/` — полезные скрипты и конфигурации (nmap, hydra, metasploit rc).
- `results/` — примерные логи и выводы (заглушки и образцы).
- `docs/` — изображения-визуализации (заглушки Wireshark и скриншоты результатов).

> ВНИМАНИЕ: материалы — учебные. Не запускайте скрипты против чужих систем. Используйте только локальные тестовые среды (Metasploitable, ваши VM).
