# Отчёт по практической работе №2 — готовый пакет для загрузки в GitHub

Содержит:
- Полный отчёт в формате Markdown (`REPORT.md`) — оформлен по примеру файла, загруженного пользователем.
- Отдельные файлы-отчёты по каждому действию/этапу в папке `actions/` (архитектура, алгоритм, инструментарий, тестирование).
- Исходный код C++ в `src/main.cpp`.
- Скрипт тестирования `tests/run_tests.sh` и набор входных данных `tests/test_cases.txt`.
- Лицензия MIT, .gitignore.

Файлы готовы к прямой загрузке в репозиторий GitHub.
