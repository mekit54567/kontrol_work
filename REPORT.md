# Общий отчёт по практическим занятиям — Информационная безопасность

**Краткое описание:**  
В этом наборе подготовлены отчёты по трём практическим заданиям: сканирование (Nmap), подбор SSH (Metasploit), подбор HTTP Basic (Hydra). Все отчёты без персональных данных, с подробными командами, логами и рекомендациями.

## Содержание
1. Скaнирование сети и анализ уязвимостей — `actions/01_NMAP_Scan_Report.md`  
2. Brute-force SSH с Metasploit — `actions/02_SSH_BruteForce_Report.md`  
3. HTTP Basic Auth: поиск порта и подбор с Hydra — `actions/03_HTTP_BasicAuth_Report.md`

## Как пользоваться
1. Распакуйте репозиторий.  
2. Прочитайте отчёты в `actions/`.  
3. Запустите скрипты в безопасной учебной среде (см. `scripts/`), при необходимости отредактируйте IP-адреса и пути к файлам с паролями.

## Ограничения
- По этическим причинам в `results/` содержатся демонстрационные логи и результаты-заглушки.  
- Скрипты готовы к запуску только в изолированной лабораторной сети (VM).
